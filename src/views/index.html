Overview

The front-end toolkit is a highly-modular design system for building web pages. It contains a collection of components and structures that can be assembled into more complex page layouts.

This guide contains real working examples, code snippets, documentation, and style guidelines.

Start by adding the following files to your page:

Main DUX Aegon styles/scripts libraries:

<link rel="stylesheet" href="http://localhost:3000/styles/aegon-library.css">
<script src="http://localhost:3000/scripts/aegon-library.js"></script>

Widgets, tools and additional styles library:

<link rel="stylesheet" href="http://localhost:3000/styles/toolkit.css">

Tools and additional scripts under ./src/asset/scripts/**:

<script src="http://localhost:3000/scripts/toolkit.js"></script>

Change http://localhost:3000 with your host or source host serving these files.
